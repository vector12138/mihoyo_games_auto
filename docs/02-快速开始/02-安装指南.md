# 安装指南

## 📥 获取代码

### 方法一：Git 克隆（推荐）
```bash
# 克隆仓库
git clone https://github.com/vector12138/mihoyo_games_auto.git

# 进入项目目录
cd mihoyo_games_auto
```

### 方法二：下载 ZIP 文件
1. 访问 [GitHub 仓库](https://github.com/vector12138/mihoyo_games_auto)
2. 点击 "Code" → "Download ZIP"
3. 解压到合适目录

### 方法三：使用 Git 客户端
1. 安装 Git 客户端（如 GitKraken、SourceTree）
2. 克隆仓库：`https://github.com/vector12138/mihoyo_games_auto.git`
3. 选择本地目录

## 🐍 Python 环境设置

### 1. 检查 Python 版本
```bash
# 检查 Python 是否安装
python --version
# 或
python3 --version

# 应该显示 Python 3.9 或更高版本
```

### 2. 安装 Python（如未安装）
**Windows**:
1. 访问 [python.org](https://www.python.org/downloads/)
2. 下载 Python 3.9+ 安装程序
3. 安装时勾选 "Add Python to PATH"

**注意**: 本项目仅支持 Windows 系统，不支持 Linux 或 macOS。

### 3. 创建虚拟环境
```bash
# 在项目目录中
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
```

**激活后提示符变化**:
```
(venv) C:\path\to\mihoyo_games_auto>
```

## 📦 安装依赖

### 1. 基本依赖安装
```bash
# 确保在虚拟环境中
pip install -r requirements.txt
```

### 2. 可选依赖安装
```bash
# 开发依赖（可选）
pip install black flake8 mypy pytest

# GPU 加速（如有 NVIDIA 显卡）
pip install paddlepaddle-gpu
```

### 3. 验证安装
```bash
# 检查关键包是否安装成功
python -c "import paddleocr; import pyautogui; import opencv-python; print('✅ 依赖安装成功')"
```

## 🔧 Windows 特定配置

### 1. 安装 Visual C++ Redistributable
1. 下载 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)
2. 运行安装程序
3. 重启计算机（如提示）

### 2. 配置 pywin32
```bash
# 安装后可能需要运行 post-install 脚本
python -m pip install pywin32

# Windows 上可能需要管理员权限运行
python Scripts/pywin32_postinstall.py -install
```

### 3. 防病毒软件设置
1. 将项目目录添加到防病毒软件白名单
2. 允许 Python 解释器运行
3. 允许自动化脚本执行

## 🎮 游戏配置准备

### 1. 游戏安装
1. 安装原神和/或绝区零
2. 确保游戏可以正常运行
3. 完成首次登录和设置

### 2. 游戏设置调整
**显示设置**:
- 分辨率：1920x1080（推荐）
- 窗口模式：窗口化或无边框窗口
- 显示比例：16:9

**游戏设置**:
- 语言：简体中文
- 界面缩放：默认
- 画质：中等或以上

### 3. 账户设置
1. 启用记住密码功能
2. 设置自动登录
3. 确保账户安全

## ⚙️ 项目配置

### 1. 复制配置文件模板
```bash
# 复制示例配置文件
copy config.example.yaml config.yaml
# 或
cp config.example.yaml config.yaml
```

### 2. 编辑配置文件
打开 `config.yaml`，根据注释修改配置：

```yaml
# 全局配置
global:
  use_gpu: true                  # 是否使用GPU加速OCR
  max_retries: 3                 # 失败重试次数
  retry_delay: 1                 # 重试间隔（秒）

# 游戏配置
games:
  genshin:
    window_title: "原神"         # 游戏窗口标题
    enabled: true                # 是否启用
  zzz:
    window_title: "绝区零"       # 游戏窗口标题
    enabled: true                # 是否启用

# Telegram通知配置
notification:
  telegram:
    enabled: false               # 是否启用Telegram通知
    token: ""                    # Telegram Bot Token
    chat_id: ""                  # 聊天ID
```

### 3. 配置环境变量（可选）
```bash
# Windows (PowerShell)
$env:MIHOYO_GAMES_AUTO_CONFIG = "C:\path\to\config.yaml"

# 注意：本项目仅支持Windows系统
```

## 📱 Telegram 机器人设置（可选）

### 1. 创建 Telegram Bot
1. 打开 Telegram，搜索 `@BotFather`
2. 发送 `/newbot` 创建新机器人
3. 按照提示设置名称和用户名
4. 保存 Bot Token

### 2. 获取 Chat ID
1. 向你的机器人发送消息
2. 访问：`https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates`
3. 查找 `chat.id` 字段

### 3. 更新配置
```yaml
notification:
  telegram:
    enabled: true
    token: "YOUR_BOT_TOKEN"
    chat_id: "YOUR_CHAT_ID"
```

## 🐛 故障排除

```bash

# 使用国内镜像源

# 逐个安装依赖
pip install paddlepaddle
pip install paddleocr
# ... 其他依赖
```

### 常见问题 2：Python 版本问题
```bash
# 检查 Python 版本
python --version

# 如果版本不对，指定 Python 3
python3 -m venv venv
python3 -m pip install -r requirements.txt
```

### 常见问题 3：权限问题
```bash
# Windows：以管理员身份运行
# 右键点击命令行 → 以管理员身份运行

# 注意：本项目仅支持Windows系统
```

### 常见问题 4：GPU 相关错误
