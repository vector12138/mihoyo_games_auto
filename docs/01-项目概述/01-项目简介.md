# 项目简介

## 🎯 项目目标

米哈游游戏自动化工具是一个专门为米哈游旗下游戏（原神、绝区零等）设计的自动化任务执行工具。项目旨在帮助玩家自动完成日常重复性任务，节省时间，同时提供稳定可靠的自动化体验。

## 🏗️ 技术架构

### 核心设计理念
1. **模块化设计** - 各功能模块独立，便于维护和扩展
2. **配置驱动** - 通过配置文件控制行为，无需修改代码
3. **错误恢复** - 内置重试机制和错误处理
4. **Windows专用** - 仅支持 Windows 系统，不兼容 Linux 或 macOS

### 技术栈
- **编程语言**: Python 3.9+
- **OCR引擎**: PaddleOCR (基于 PaddlePaddle)
- **图像处理**: OpenCV, Pillow
- **自动化控制**: pynput
- **Windows API**: pywin32
- **通知系统**: python-telegram-bot
- **配置管理**: YAML

## 📦 主要功能

### 1. 日常任务自动化
- 自动登录游戏
- 完成每日委托
- 领取日常奖励
- 树脂管理

### 2. 智能识别系统
- 基于 OCR 的文本识别
- 图像匹配和定位
- 状态检测和判断

### 3. 多应用管理
- 同时管理游戏客户端和辅助工具
- 窗口切换和焦点管理
- 进程监控和控制

### 4. 远程控制
- 远程唤醒功能 (WOL)
- Telegram 通知和远程控制
- 状态监控和报告

### 5. 系统集成
- 自动关机功能
- 错误日志记录
- 性能监控

## 🔄 发展历程

### 版本演进
- **v1.x** - 基于 AutoHotkey 的早期版本
- **v2.0** - 全面重构，全 Python 实现
- **v2.1** - 添加多应用支持和远程功能

### 技术改进
1. **性能提升** - 使用 BitBlt 高速截图替代传统截图
2. **稳定性增强** - 引入重试机制和错误恢复
3. **扩展性优化** - 模块化设计，易于添加新游戏
4. **用户体验** - 添加 Telegram 通知和远程控制
